# 💡 macOS

| Pro Tip | Note |
| --- | --- |
| Select multiple contiguous files at once in Finder | + Method 1: `Cmd`+`2` (List View) &rarr; Shift click for your selection &rarr; `Cmd`+`1` (Icon View) <br>+ Method 2: Drag a rectangle around files |
| [Batch rename multiple files](https://stackoverflow.com/a/53321208) |  |
| Right-click a folder in Finder on macOS, you will get an "Import from iPhone/iPad" option. Click that and your iPhone/iPad will open a scanner app which lets you "Take Photo", "Scan Documents", and "Add Sketch." |  |
| Copy and move files&folders in Finder | First copy (`Cmd` + `C`), then move (`Opt` + `Cmd` + `V`) |
