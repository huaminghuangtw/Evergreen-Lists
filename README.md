## About

This repo maintains evergreen lists that I curated over time. Each list is available in three formats: **Google Sheets**, **JSON**, and **Markdown**, feel free to use them in the way that suits you best!

Contributions are welcome! If you have ideas to expand or improve these lists, please [open an issue](https://github.com/huaminghuangtw/Evergreen-Lists/issues/new) or [submit a pull request](https://github.com/huaminghuangtw/Evergreen-Lists/compare).
